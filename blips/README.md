# Blips
## [DEUTSCH]

### Beschreibung
Dieses Skript fuegt automatisch Kartenmarkierungen (Blips) fuer wichtige Orte hinzu. Alle Blips werden ueber eine zentrale Konfigurationsdatei verwaltet.

### Installation
1. **Entpacken**: Entpacke die `blips.zip` Datei.
2. **Kopieren**: Schiebe den extrahierten Ordner `blips` in dein FiveM `resources` Verzeichnis.
3. **Aktivieren**: oeffne deine `server.cfg` und fuege dort die Zeile `ensure blips` hinzu.
4. **Server starten**: Starte deinen Server neu oder nutze den Befehl `refresh` und `start blips` in der Konsole.

### Konfiguration
Du kannst neue Markierungen in der Datei `config/config.lua` hinzufuegen. Dort lassen sich Koordinaten, Icons (Sprites), Farben und Namen anpassen.

---

## [ENGLISH]

### Description
This script automatically adds map markers (blips) for important locations to your server. All blips are managed through a central configuration file.

### Installation
1. **Extract**: Unzip the `blips.zip` file.
2. **Copy**: Move the extracted `blips` folder into your FiveM `resources` directory.
3. **Activate**: Open your `server.cfg` and add the line `ensure blips`.
4. **Start Server**: Restart your server or use the command `refresh` and `start blips` in the console.

### Configuration
You can add new markers in the `config/config.lua` file. Coordinates, icons (sprites), colors, and names can be customized there.
